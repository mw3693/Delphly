# Hauptthema: Die Familie

---

## Sprachniveau: A1

### Unterthema: Familienmitglieder vorstellen

#### Unter-Unterthema: Kernfamilie (Eltern, Geschwister, Kinder)

**Kommunikative Funktion: eine Person vorstellen**
- Pattern: Das ist [Familienmitglied].
  - Variablen — Familienmitglied: mein Vater / meine Mutter / mein Bruder / meine Schwester / mein Sohn / meine Tochter
- Pattern: Ich möchte dir [Familienmitglied] vorstellen.
  - Variablen — Familienmitglied: meinen Vater / meine Mutter / meinen Bruder / meine Schwester

**Kommunikative Funktion: Existenz/Besitz von Familienmitgliedern ausdrücken**
- Pattern: Ich habe [Quantity] [Familienmitglied].
  - Variablen — Quantity: einen / eine / zwei / drei / keine
  - Variablen — Familienmitglied: Bruder / Schwester / Kind / Kinder / Geschwister
- Pattern: Ich habe keine [Familienmitglied].
  - Variablen — Familienmitglied: Geschwister / Kinder

**Kommunikative Funktion: nach Familienmitgliedern fragen**
- Pattern: Hast du [Familienmitglied]?
  - Variablen — Familienmitglied: Geschwister / Kinder / einen Bruder / eine Schwester
- Pattern: Wie viele [Familienmitglied] hast du?
  - Variablen — Familienmitglied: Geschwister / Kinder / Brüder / Schwestern

**Kommunikative Funktion: Namen nennen**
- Pattern: [Familienmitglied] heißt [Name].
  - Variablen — Familienmitglied: Mein Vater / Meine Mutter / Mein Bruder / Meine Schwester
  - Variablen — Name: Ahmed / Sara / Karim / Layla

**Kommunikative Funktion: Alter angeben**
- Pattern: [Familienmitglied] ist [Age] Jahre alt.
  - Variablen — Familienmitglied: Mein Vater / Meine Mutter / Mein Bruder / Meine Schwester
  - Variablen — Age: zehn / zwanzig / vierzig / fünfzig / sechzig

**Kommunikative Funktion: Beruf eines Familienmitglieds nennen**
- Pattern: [Familienmitglied] ist [Beruf] von Beruf.
  - Variablen — Familienmitglied: Mein Vater / Meine Mutter / Mein Bruder
  - Variablen — Beruf: Lehrer / Arzt / Ingenieurin / Verkäufer / Hausfrau

#### Unter-Unterthema: Erweiterte Familie (Großeltern, Onkel, Tante, Cousin/Cousine)

**Kommunikative Funktion: eine Person vorstellen**
- Pattern: Das ist [Familienmitglied] von mir.
  - Variablen — Familienmitglied: mein Onkel / meine Tante / mein Cousin / meine Cousine / mein Opa / meine Oma

**Kommunikative Funktion: Verwandtschaftsverhältnis erklären**
- Pattern: [Person] ist [Verwandtschaft] von [Bezugsperson].
  - Variablen — Person: Sara / Karim / Layla
  - Variablen — Verwandtschaft: die Schwester / der Bruder / die Tochter / der Sohn
  - Variablen — Bezugsperson: meinem Vater / meiner Mutter

**Kommunikative Funktion: nach Verwandtschaft fragen**
- Pattern: Wer ist [Person] für dich?
  - Variablen — Person: er / sie / diese Frau / dieser Mann
- Pattern: Ist [Person] deine [Verwandtschaft]?
  - Variablen — Person: er / sie
  - Variablen — Verwandtschaft: Tante / Onkel / Cousine / Cousin

#### Unter-Unterthema: Familienstand

**Kommunikative Funktion: Familienstand angeben**
- Pattern: [Familienmitglied] ist [Familienstand].
  - Variablen — Familienmitglied: Mein Bruder / Meine Schwester / Meine Tante
  - Variablen — Familienstand: verheiratet / ledig / geschieden / verwitwet / verlobt

**Kommunikative Funktion: nach Familienstand fragen**
- Pattern: Bist du [Familienstand]?
  - Variablen — Familienstand: verheiratet / ledig / verlobt

### Unterthema: Familie beschreiben

#### Unter-Unterthema: Aussehen

**Kommunikative Funktion: Aussehen beschreiben**
- Pattern: [Familienmitglied] hat [Attribut].
  - Variablen — Familienmitglied: Mein Bruder / Meine Schwester / Mein Vater
  - Variablen — Attribut: kurze Haare / lange Haare / braune Augen / einen Bart
- Pattern: [Familienmitglied] ist [Adjektiv].
  - Variablen — Familienmitglied: Mein Vater / Meine Mutter
  - Variablen — Adjektiv: groß / klein / schlank / dünn

#### Unter-Unterthema: Charakter

**Kommunikative Funktion: Charakter beschreiben**
- Pattern: [Familienmitglied] ist sehr [Charaktereigenschaft].
  - Variablen — Familienmitglied: Mein Vater / Meine Mutter / Mein Bruder / Meine Schwester
  - Variablen — Charaktereigenschaft: nett / freundlich / streng / lustig / ruhig

**Kommunikative Funktion: Gefühle gegenüber Familie ausdrücken**
- Pattern: Ich liebe [Familienmitglied] sehr.
  - Variablen — Familienmitglied: meine Familie / meinen Vater / meine Mutter / meine Geschwister
- Pattern: Ich mag [Familienmitglied].
  - Variablen — Familienmitglied: meinen Bruder / meine Schwester / meine Oma

#### Unter-Unterthema: Haustiere in der Familie

**Kommunikative Funktion: Haustiere nennen**
- Pattern: Wir haben [Quantity] [Haustier] zu Hause.
  - Variablen — Quantity: einen / eine / zwei
  - Variablen — Haustier: Hund / Katze / Vogel / Fisch

### Unterthema: Zusammenleben

#### Unter-Unterthema: Wohnort der Familie

**Kommunikative Funktion: Wohnort angeben**
- Pattern: [Familienmitglied] wohnt in [Ort].
  - Variablen — Familienmitglied: Meine Familie / Meine Großeltern / Meine Tante
  - Variablen — Ort: Kairo / Berlin / einem Dorf / der Stadt

**Kommunikative Funktion: Zusammenleben beschreiben**
- Pattern: Ich wohne mit [Familienmitglied] zusammen.
  - Variablen — Familienmitglied: meinen Eltern / meiner Familie / meinem Bruder

#### Unter-Unterthema: Familienbesuche

**Kommunikative Funktion: einen Besuch ankündigen**
- Pattern: Wir besuchen [Familienmitglied] [Zeitpunkt].
  - Variablen — Familienmitglied: meine Großeltern / meine Tante / meine Cousins
  - Variablen — Zeitpunkt: heute / am Wochenende / jeden Freitag

**Kommunikative Funktion: einen Besuch beschreiben**
- Pattern: [Familienmitglied] besucht uns oft.
  - Variablen — Familienmitglied: Meine Oma / Mein Onkel / Meine Tante

### Unterthema: Höflichkeit und einfache Interaktionen in der Familie

#### Unter-Unterthema: um Erlaubnis bitten

**Kommunikative Funktion: um Erlaubnis bitten**
- Pattern: Darf ich [Aktivität]?
  - Variablen — Aktivität: zu meiner Oma gehen / mit meinem Bruder spielen / bei meiner Tante schlafen
- Pattern: Kann ich bitte [Aktivität]?
  - Variablen — Aktivität: meine Cousine besuchen / mit meinem Vater sprechen

**Kommunikative Funktion: Erlaubnis geben oder verweigern**
- Pattern: Ja, du darfst [Aktivität].
  - Variablen — Aktivität: gehen / spielen / mitkommen
- Pattern: Nein, du darfst nicht [Aktivität].
  - Variablen — Aktivität: so spät nach Hause kommen / allein gehen

#### Unter-Unterthema: sich bedanken

**Kommunikative Funktion: sich bedanken (informell)**
- Pattern: Danke für [Sache], [Familienmitglied]!
  - Variablen — Sache: das Geschenk / die Hilfe / das Essen
  - Variablen — Familienmitglied: Mama / Papa / Oma

**Kommunikative Funktion: sich bedanken (höflich/formell)**
- Pattern: Vielen Dank, dass Sie [Handlung].
  - Variablen — Handlung: mir geholfen haben / auf die Kinder aufgepasst haben

#### Unter-Unterthema: einfache Vergleiche zwischen Familienmitgliedern

**Kommunikative Funktion: zwei Personen vergleichen**
- Pattern: [Familienmitglied1] ist [Adjektiv] als [Familienmitglied2].
  - Variablen — Familienmitglied1: Mein Bruder / Meine Schwester
  - Variablen — Adjektiv: größer / älter / jünger / kleiner
  - Variablen — Familienmitglied2: ich / meine Schwester / mein Vater
- Pattern: [Familienmitglied1] sieht [Familienmitglied2] ähnlich.
  - Variablen — Familienmitglied1: Mein Sohn
  - Familienmitglied2: seinem Vater / seiner Mutter

#### Unter-Unterthema: nach jemandes Familie fragen (höflich)

**Kommunikative Funktion: höflich nach der Familie fragen (Sie-Form)**
- Pattern: Haben Sie [Familienmitglied]?
  - Variablen — Familienmitglied: Kinder / Geschwister / eine große Familie
- Pattern: Wie geht es Ihrer [Familienmitglied]?
  - Variablen — Familienmitglied: Familie / Frau / Mutter

### Unterthema: Essen mit der Familie

#### Unter-Unterthema: gemeinsame Mahlzeiten

**Kommunikative Funktion: gemeinsame Mahlzeiten beschreiben**
- Pattern: Wir essen [Mahlzeit] zusammen.
  - Variablen — Mahlzeit: Frühstück / Mittagessen / Abendessen
- Pattern: [Familienmitglied] kocht [Essen] für die Familie.
  - Variablen — Familienmitglied: Meine Mutter / Meine Oma
  - Variablen — Essen: Reis / Suppe / Kuchen

## Sprachniveau: A2

### Unterthema: Familienalltag

#### Unter-Unterthema: Tagesablauf der Familie

**Kommunikative Funktion: tägliche Routinen beschreiben**
- Pattern: [Familienmitglied] [Aktivität] jeden Tag / jeden Morgen / jeden Abend.
  - Variablen — Familienmitglied: Meine Mutter / Mein Vater / Meine Geschwister
  - Variablen — Aktivität: kocht / arbeitet / macht das Frühstück / bringt die Kinder zur Schule

**Kommunikative Funktion: Häufigkeit ausdrücken**
- Pattern: Wir [Aktivität] [Häufigkeit].
  - Variablen — Aktivität: essen zusammen / treffen uns / telefonieren
  - Variablen — Häufigkeit: jeden Tag / einmal pro Woche / oft / selten / manchmal

**Kommunikative Funktion: Aufgabenverteilung beschreiben**
- Pattern: [Familienmitglied] kümmert sich um [Aufgabe].
  - Variablen — Familienmitglied: Meine Mutter / Mein Vater / Meine Schwester
  - Variablen — Aufgabe: den Haushalt / die Kinder / das Kochen / die Einkäufe

#### Unter-Unterthema: Hilfe im Haushalt

**Kommunikative Funktion: um Hilfe bitten**
- Pattern: Kannst du mir bei [Aufgabe] helfen?
  - Variablen — Aufgabe: dem Abwasch / dem Aufräumen / den Hausaufgaben

**Kommunikative Funktion: Hilfe anbieten**
- Pattern: Ich helfe dir gern bei [Aufgabe].
  - Variablen — Aufgabe: dem Kochen / dem Putzen / den Einkäufen

**Kommunikative Funktion: über Verpflichtungen sprechen**
- Pattern: Ich muss [Aufgabe] machen, bevor ich [Aktivität] kann.
  - Variablen — Aufgabe: mein Zimmer aufräumen / den Müll rausbringen
  - Variablen — Aktivität: ausgehen / fernsehen

### Unterthema: Familienereignisse und Feiern

#### Unter-Unterthema: Feste und Feiern

**Kommunikative Funktion: über vergangene Familienereignisse berichten**
- Pattern: Letztes Jahr haben wir [Ereignis] gefeiert.
  - Variablen — Ereignis: den Geburtstag von meiner Mutter / die Hochzeit meiner Schwester / Weihnachten

**Kommunikative Funktion: zukünftige Familienpläne ausdrücken**
- Pattern: Nächstes Jahr werden wir [Ereignis] feiern.
  - Variablen — Ereignis: den 60. Geburtstag meines Vaters / die Hochzeit meines Bruders
- Pattern: Wir planen, [Aktivität] zu machen.
  - Variablen — Aktivität: eine Familienreise / ein großes Fest / ein Familientreffen

**Kommunikative Funktion: einladen**
- Pattern: Möchtest du zu [Ereignis] kommen?
  - Variablen — Ereignis: meinem Geburtstag / unserem Familienfest / der Feier

**Kommunikative Funktion: gratulieren**
- Pattern: Herzlichen Glückwunsch zu [Anlass]!
  - Variablen — Anlass: deinem Geburtstag / eurer Hochzeit / der Geburt des Babys

#### Unter-Unterthema: Familientraditionen

**Kommunikative Funktion: Traditionen beschreiben**
- Pattern: In meiner Familie ist es Tradition, [Aktivität] zu machen.
  - Variablen — Aktivität: sich an Feiertagen zu treffen / gemeinsam zu kochen / Geschichten zu erzählen

#### Unter-Unterthema: Namensgebung

**Kommunikative Funktion: die Wahl eines Babynamens beschreiben**
- Pattern: Wir haben unser Baby [Name] genannt, weil [Grund].
  - Variablen — Name: Layla / Ahmed / Sara
  - Variablen — Grund: der Name in der Familie Tradition hat / er uns sehr gefällt

**Kommunikative Funktion: über den Nachnamen nach der Heirat sprechen**
- Pattern: Nach der Hochzeit hat [Person] [Handlung].
  - Variablen — Person: meine Schwester / meine Frau
  - Variablen — Handlung: den Nachnamen ihres Mannes angenommen / ihren eigenen Nachnamen behalten

### Unterthema: Familie und Reisen

#### Unter-Unterthema: Familienurlaub

**Kommunikative Funktion: über einen vergangenen Urlaub berichten**

- Pattern: Wir streiten uns, weil [Grund].
  - Variablen — Grund: er nie hilft / sie immer zu spät kommt / wir unterschiedliche Meinungen haben

### Unterthema: Familie unterstützen und reagieren

#### Unter-Unterthema: einfache Ratschläge und Vorschläge

**Kommunikative Funktion: einen Ratschlag geben**
- Pattern: Du solltest [Aktivität].
  - Variablen — Aktivität: mit deiner Mutter sprechen / dich bei ihm entschuldigen / mehr Zeit mit der Familie verbringen
- Pattern: Ich würde an deiner Stelle [Aktivität].
  - Variablen — Aktivität: ihn anrufen / ihr helfen / mit ihr reden

**Kommunikative Funktion: einen Vorschlag machen**
- Pattern: Wie wäre es, wenn wir [Aktivität]?
  - Variablen — Aktivität: zusammen kochen / Oma besuchen / einen Familienausflug machen
- Pattern: Sollen wir [Aktivität]?
  - Variablen — Aktivität: zusammen essen gehen / die Familie einladen

#### Unter-Unterthema: Zufriedenheit und Unzufriedenheit ausdrücken

**Kommunikative Funktion: sich beschweren (einfach)**
- Pattern: Ich bin nicht zufrieden, weil [Grund].
  - Variablen — Grund: mein Bruder nie aufräumt / niemand mir hilft / du immer zu spät kommst
- Pattern: Es stört mich, dass [Grund].
  - Variablen — Grund: er so laut Musik hört / sie nie anruft

**Kommunikative Funktion: loben**
- Pattern: [Familienmitglied] hat [Leistung] wirklich gut gemacht.
  - Variablen — Familienmitglied: Mein Sohn / Meine Tochter / Mein Bruder
  - Variablen — Leistung: die Prüfung / das Essen / die Arbeit im Garten
- Pattern: Ich bin sehr stolz auf [Familienmitglied].
  - Variablen — Familienmitglied: meinen Sohn / meine Tochter / meinen Bruder

#### Unter-Unterthema: Familienfotos beschreiben

**Kommunikative Funktion: ein Familienfoto beschreiben**
- Pattern: Auf dem Foto [Aktivität] [Familienmitglied].
  - Variablen — Aktivität: lächelt / sitzt / steht neben meiner Mutter

  - Variablen — Ereignis: meiner Hochzeit / unserem letzten Urlaub / dem Geburtstag meiner Oma

### Unterthema: Erziehung und Werte

#### Unter-Unterthema: Erziehungsstile

**Kommunikative Funktion: Erziehungsmethoden beschreiben**
- Pattern: Meine Eltern haben mich [Adverb] erzogen.
  - Variablen — Adverb: streng / locker / liebevoll / traditionell
- Pattern: In meiner Familie legt man Wert auf [Wert].
  - Variablen — Wert: Respekt / Disziplin / Ehrlichkeit / Bildung

**Kommunikative Funktion: Meinung zur Erziehung äußern**
- Pattern: Ich finde, dass Eltern [Aussage] sollten.
  - Variablen — Aussage: ihren Kindern mehr Freiheit geben / klare Regeln setzen / mit gutem Beispiel vorangehen
- Pattern: Meiner Meinung nach ist es wichtig, dass Kinder [Aussage].
  - Variablen — Aussage: lernen, Verantwortung zu übernehmen / ihre Meinung sagen dürfen

**Kommunikative Funktion: zustimmen**
- Pattern: Da stimme ich dir zu, [Aussage].
  - Variablen — Aussage: Erziehung sollte auf Vertrauen basieren / Kinder brauchen klare Grenzen

**Kommunikative Funktion: widersprechen**
- Pattern: Ich sehe das anders, [Gegenaussage].
  - Variablen — Gegenaussage: zu viel Strenge schadet den Kindern / Freiheit ohne Regeln funktioniert nicht

#### Unter-Unterthema: schulische Unterstützung

**Kommunikative Funktion: über Unterstützung bei der Schule sprechen**
- Pattern: [Familienmitglied] hilft [Person] bei [Fach].
  - Variablen — Familienmitglied: Meine Mutter / Mein älterer Bruder
  - Variablen — Person: meinem Sohn / meiner Tochter / mir
  - Variablen — Fach: den Hausaufgaben / der Mathematik / dem Lernen
